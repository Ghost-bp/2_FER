| 参数 | 值 |

## 优化路径

```
基线 (68.80%)
  → L1: ImageNet 预训练 (+11.0pp to 79.84%)
  → L1: ResNet-34 (+6.8pp to 86.64%)
  → L1: ResNet-50 (+1.8pp to 88.42%)
  → L2: Cosine 调度器 (+0.8pp to 89.24%)
  → L4: Geometric 增强 (+1.5pp to 90.74%)
  → L5: div=3 损失权重 (+1.5pp to 92.22%)
  → L6: epochs=80 (+0.5pp to 92.69%)
  ─────────────────────────
  总计: +17.84pp (vs 师兄基线 74.38%)*
```

*\* 基线差异因 6 类 vs 7 类、代码修复等因素，优化路径内部的相对提升是可靠的。*

## 关键发现

1. **ImageNet >> MSCeleb**: 即使 MSCeleb 是人脸预训练，ImageNet 的多样性带来 +11pp 提升
2. **Backbone 越深越好**: RN18→34→50 持续提升，边际递减 (+11→+6.8→+1.8)
3. **Cosine 调度器最优**: 比 ExpLR 精度更高、方差减半
4. **默认 LR/WD 恰是最优**: lr=2e-4, wd=1e-4 无需调整
5. **几何增强有效**: Rotation + Affine 带来 +1.5pp
6. **颜色增强致命**: ColorJitter + Normalize = 23.52%（禁用！）
7. **降低多样性约束**: div=3 > div=5 > div=7，弱约束更利于主任务
8. **更长训练有益**: ep=80 > ep=60，早停自动控制实际轮数
