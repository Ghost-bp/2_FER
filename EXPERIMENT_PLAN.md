# CAFE 模型轻量化实验 — 主控追踪

> **每次会话必读此文件。** 记录了整体计划、当前进度、下一步操作。

## 实验目标

分析 CAFE 模型各组件参数量（CLIP ~151M 占 93%），通过减少 CLIP 层数/替换为更小模型/移除 CLIP 等方式降低参数量，对比精度和效率。

## Git 分支体系

```
main                         ← 原始代码（不改动）
├── exp/param-analysis       ← 实验 0：参数量分析
├── exp/baseline-kfold       ← 实验 1：KFold+YOLO 基线
├── exp/clip-9               ← 实验 2：CLIP-9 层
├── exp/clip-6               ← 实验 3：CLIP-6 层
├── exp/clip-3               ← 实验 4：CLIP-3 层
├── exp/resnet-only          ← 实验 5：纯 ResNet-18
├── exp/mobilenet            ← 实验 6：MobileNetV3 替代
└── exp/clip-text            ← 实验 7：CLIP 文本辅助
```

## 流水线流程

```
1. git checkout <实验分支>
2. 运行训练
3. training-logger → 结构化训练记录
4. training-analyzer → 可视化图表 + 分析
5. git commit + git push origin <分支>
6. 更新本文件进度
7. 更新 LEARNING_GUIDE.md
8. 冷却 10 分钟
9. 进入下一个实验
```
